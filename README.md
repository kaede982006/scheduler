# scheduler

GNU GCC / Linux 환경의 CLI 스케줄러.  
달력 기반 **1형식**과 시간 순 **2형식** 두 가지 뷰를 제공하며,  
텍스트 라인 및 트리 형식으로 일정을 출력한다.

---

## 빌드

```bash
make          # 또는 make all
make install  # ~/.local/bin/scheduler 에 설치
make clean    # 오브젝트 파일 및 바이너리 제거
```

> **요구사항**: gcc, GNU make

---

## 사용법

```
scheduler -d [YYYY-MM]        1형식: 월간 달력 뷰 (읽기)
scheduler -u [YYYY-MM-DD]     2형식: 일간 시간 뷰 (읽기)
scheduler -m [YYYY-MM-DD]     2형식: 편집 모드     ※ 단독 옵션
```

- 날짜 인자를 생략하면 **오늘 날짜**를 기준으로 동작한다.
- `-m` 은 다른 옵션(`-d`, `-u`)과 **함께 사용할 수 없다**.

### 예시

```bash
# 이번 달 전체 일정 조회
scheduler -d

# 특정 월 조회
scheduler -d 2026-03

# 오늘 일정 조회
scheduler -u

# 특정 날 조회
scheduler -u 2026-03-09

# 특정 날 편집 모드 진입
scheduler -m 2026-03-09
```

## 출력 형식

두 형식 모두 **텍스트 라인** → **트리** 순으로 연속 출력된다.

### 1형식 (`-d`) — 월간 달력 뷰

일마다 등록된 일정을 시간순으로 나열한다.

```
◆ [1형식 / 텍스트 라인]  2026년 3월
────────────────────────────────────
2026-03-09 (월)
  [09:00]  팀 스탠드업 미팅
  [14:00]  아키텍처 설계 회의

◆ [1형식 / 트리]  2026년 3월
2026-03
├── 09일 (월) [2건]
│   ├── [09:00]  팀 스탠드업 미팅
│   └── [14:00]  아키텍처 설계 회의
└── 15일 (일) [1건]
    └── [09:00]  리눅스마스터 시험
```

### 2형식 (`-u`, `-m`) — 일간 시간 뷰

시간대별로 그룹화하여 출력한다.

```
◆ [2형식 / 텍스트 라인]  2026년 3월 9일 (월)
────────────────────────────────────
  09:00 대
    [09:00]  팀 스탠드업 미팅
  14:00 대
    [14:00]  아키텍처 설계 회의
    [14:30]  DB 스키마 검토

◆ [2형식 / 트리]  2026년 3월 9일 (월)
2026-03-09
├── 09:00 [1건]
│   └── [09:00]  팀 스탠드업 미팅
└── 14:00 [2건]
    ├── [14:00]  아키텍처 설계 회의
    └── [14:30]  DB 스키마 검토
```

## 편집 명령어 (`-m`)

편집 모드 진입 후 사용할 수 있는 대화형 명령어:

| 명령어 | 설명 |
|--------|------|
| `add HH:MM <제목>` | 일정 추가 |
| `del <ID>` | 일정 삭제 |
| `edit <ID> HH:MM <제목>` | 일정 수정 |
| `list` | 현재 날 목록 및 ID 확인 |
| `help` | 도움말 출력 |
| `quit` / `exit` | 저장 후 종료 |

`del` / `edit` 시 필요한 ID는 `list` 로 확인한다.  
종료 시 변경 사항이 자동으로 저장된다.
